# Announcing Strada

**Author:** Jay Ohms, Mobile Team Lead
**Published:** September 20, 2023
**Source:** <https://dev.37signals.com/announcing-strada/>

*Open sourcing a new member of the Hotwire family.*

---

I’m happy to announce the official launch of [Strada](https://strada.hotwired.dev), the long-awaited member of the [Hotwire](https://hotwired.dev) family. Strada enables you to create fully native controls in your hybrid mobile apps, driven by the web. Build web components and native components that work together in `WebView` screens to elevate your [Turbo Native](https://turbo.hotwired.dev/handbook/native) apps to the next level.

![Strada](https://dev.37signals.com/assets/images/announcing-strada/strada.png)

Create fully native controls, driven by your web app

We’ve been using Strada to build the [HEY](https://www.hey.com) mobile apps for the past 3 years and it’s made a huge difference in the level of refinement that we’ve been able to deliver in our [iOS](https://apps.apple.com/lt/app/hey-email/id1506603805) and [Android](https://play.google.com/store/apps/details?id=com.basecamp.hey) apps. It’s the bridge between native controls and your web app.

We’re excited to share this latest open source project with you and we’ve provided thorough documentation, installation instructions, demo examples, and reference guides to help you get started. You can read all the details on [strada.hotwired.dev](https://strada.hotwired.dev). Be sure to check out all the individual GitHub repos where you can browse the source code, submit issues, and create pull requests.

---

## Library repos
- [hotwired/strada-web](https://github.com/hotwired/strada-web): The JavaScript library for your web app
- [hotwired/strada-ios](https://github.com/hotwired/strada-ios): The Swift library for your iOS app
- [hotwired/strada-android](https://github.com/hotwired/strada-android): The Kotlin library for your Android app

## Demo repos
- [hotwired/turbo-native-demo](https://github.com/hotwired/turbo-native-demo): The Turbo Native web demo with Strada examples
- [hotwired/turbo-ios](https://github.com/hotwired/turbo-ios/tree/main/Demo): The Turbo iOS demo app with Strada examples
- [hotwired/turbo-android](https://github.com/hotwired/turbo-android/tree/main/demo): The Turbo Android demo app with Strada examples

## Website repo
- [hotwired/strada-site](https://github.com/hotwired/strada-site): The source for the strada.hotwired.dev site

Strada enables you to build Turbo Native apps with native controls and higher fidelity than was previously possible. Give it a try and I’m looking forward to your feedback and contributions!
